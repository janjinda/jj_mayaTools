This toolkit is a set of simple scripts tailored to provide clean, easier and more affective workflow for handling OBJ files in Maya. Thanks to the Import as blend shape options it keeps all your scene hierarchy, geometry UVs, shader assignments etc.

Why OBJ?

OBJ is basically very simple text file containing just vertex positions, face normals, UVs and group names. Therefore it is very suitable if model is needed to be passed between softwares. Probably the main destination for OBJ files is ZBrush. It is used not just for sculpting details, but even for bigger overall changes to a lot of individual geometries, thanks to it's Transpose Master.

Import

Choose one or multiple OBJ files. All those will be imported and grouped in OBJ_import_grp (unless Import as single geo is enabled). Script will name every geometry based on the file name free from all the clutter Maya usually imports with OBJs (shaders, shading groups, faceted normals, selection sets, etc.)

Import Single as bShape

Select one geometry and choose an OBJ file which will be applied as a blend shape. Controller will be automatically created with a blend value attribute. If OBJ is not a valid geometry for a blend shape, it will be grouped to OBJ_import_grp.

Import Batch as bShape

Choose an OBJ files which will be applied as blend shapes to corresponding geometries in the scene based on matching OBJ and geometry name. Controller will be automatically created with a blend value attribute. If OBJ is not a valid geometry for a blend shape, it will be grouped to OBJ_import_grp.

Import Options

Import as single geo - When enabled multiple OBJ files will be combined to a single geometry

Delete history - When enabled a blend shape controller would not be created and construction history will be deleted on blend shaped geometries

Export

Select one or multiple geometries or parent of the hierarchy. All selected geometries or geometries found in hierarchy will be exported as individual OBJs (unless Export as single OBJ is enabled) and they will be named after geometry names.

Export Options

Export as single OBJe - When enabled selected geometries or hierarchy will be exported as single OBJ file.

Force Overwrite - skips Maya's dialog when overwriting existing OBJ file
